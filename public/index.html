Nostr2Discord Bot

Status: Active

This service forwards Nostr events to Discord channels via webhooks.

For configuration, use the webhook API with your secret key.

API Endpoints:

  • GET /api/webhook-simple - Check status
  • POST /api/webhook-simple - Update configuration (requires secret)
POST /api/webhook-simple

Configure the bot with your Nostr public key and Discord webhook

Example Request 
{
  "nostrPubkey": "your_npub_or_hex_key",
  "discordWebhook": "https://discord.com/api/webhooks/...",
  "relays": "wss://relay1.com,wss://relay2.com",
  "preferredClient": "both"
}
